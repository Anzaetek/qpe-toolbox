# Contributing

Thank you for your interest in contributing!

## License

## Developer Certificate of Origin (DCO)
By contributing, you agree to the Developer Certificate of Origin 1.1 (https://developercertificate.org/).

Each commit must be signed off:
    git commit -s -m "Your change summary"

The sign-off certifies that you have the right to submit the work and license it under Apache-2.0.

## How to contribute
- Fork and create a topic branch.
- Write tests and documentation.
- Ensure lint/format pass.
- Open a Pull Request and describe changes, rationale, and any licensing notices required in NOTICE.

## Attribution
If your contribution incorporates third-party code that requires attribution, add an entry in NOTICE and preserve upstream license notices in the corresponding files.
